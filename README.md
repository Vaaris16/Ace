# [RUST] ACE 

## Installation

```bash
cargo install --git https://github.com/Vaaris16/Ace.git
```


## What is Ace 
Ace is a Rust-powered CLI tool that instantly adds popular packages to your Vite projects — no manual config, no file editing, no headaches. Just run a simple command and you're good to go.



> ⚠️ Ace is designed for Vite-based projects only.

## Usage 

```bash
ace add <PACKAGE> --framework <FRAMEWORK>
```
